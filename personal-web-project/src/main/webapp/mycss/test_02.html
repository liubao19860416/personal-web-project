上海宝成钻石加工中心有限公司是由上海工艺美术总公司、中国第一铅笔股份（集团）公司和上海钻石厂等多元投资组建的都市型企业。他不仅拥有上市股份公司雄厚的资金实力、拥有金银首饰加工生产的强大技术力量和销售网络，还拥有钻石厂深厚的钻石加工基础和一批几十年从事钻石加工业的优秀人才，因此宝成是中国钻石加工最具权威的企业之一。 宝成公司为混合型股份有限公司，设立董事会，实行董事会领导下的总经理负责制。公司目前生产的主要产品是：“宝成”牌、“上钻”牌钻石、钻饰、工业用钻及各种铂金饰品。其中“上钻”牌钻石，被授予国家金杯奖及轻工部优秀出口产品奖；“宝成”牌钻石是源于“上钻”、精益求精的新一代钻石、钻饰。 公司的发展宗旨是：“创造高品质的钻石饰品，努力成为中国乃至亚洲地区最大的钻石加工中心。”为此宝成还建立了“首饰钻琢磨工”培训基地，大力培养钻石加工人才。 宝成公司在创建过程中，得到了国内外同行的支持和合作，在进一步改革发展过程中，海宝成钻石加工中心有限公司是由上海工艺美术总公司、中国第一铅笔股份（集团）公司和上海钻石厂等多元投资组建的都市型企业。他不仅拥有上市股份公司雄厚的资金实力、拥有金银首饰加工生产的强大技术力量和销售网络，还拥有钻石厂深厚的钻石加工基础和一批几十年从事钻石加工业的优秀人才，因此宝成是中国钻石加工最具权威的企业之一。 宝成公司为混合型股份有限公司，设立董事会，实行董事会领导下的总经理负责制。公司目前生产的主要产品是：“宝成”牌、“上钻”牌钻石、钻饰、工业用钻及各种铂金饰品。其中“上钻”牌钻石，被授予国家金杯奖及轻工部优秀出口产品奖；“宝成”牌钻石是源于“上钻”、精益求精的新一代钻石、钻饰。 公司的发展宗旨是：“创造高品质的钻石饰品，努力成为中国乃至亚洲地区最大的钻石加工中心。”为此宝成还建立了“首饰钻琢磨工”培训基地，大力培养钻石加工人才。 宝成公司在创建过程中，得到了国内外同行的支持和合作，在进一步改革发展过程中，公司仍然热忱地期望同国内外同行、客商及广大消费者建立广泛的联系、交流和合作。
